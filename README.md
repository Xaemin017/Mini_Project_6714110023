# Mini_Project_6714110023

ฟีเจอร์หลัก (Key Features)
ระบบสมาชิก & พนักงาน: รองรับการสมัครสมาชิกและการเข้าสู่ระบบ (Login/Logout)

การจัดการสินค้า: เพิ่ม, แก้ไข, ลบ ข้อมูลสินค้า 

ระบบตะกร้าสินค้า: เลือกสินค้าลงตะกร้าและคำนวณราคาสินค้าอัตโนมัติ

ระบบรายงานการขาย: ออกรายงานสรุปยอดขายตามช่วงเวลา และวิเคราะห์สินค้าขายดี

การจัดการจ่ายเงิน: บันทึกข้อมูลการชำระเงินและสรุปคำสั่งซื้อ
===================================================================================

เทคโนโลยีที่ใช้ (Tech Stack)
Back-end: Node.js, Express.js

Database: MongoDB (เชื่อมต่อผ่าน Mongoose)

Template Engine: EJS (Server-side Rendering)

Authentication: Express Session (จัดการสถานะการเข้าใช้งาน)


=============================================================================



วิธีการติดตั้งและรันระบบ (Installation Guide)

1. เตรียมฐานข้อมูล (Database)
เปิดโปรแกรม MongoDB Compass

สร้างฐานข้อมูลชื่อ ITMISHOP (หรือตรวจสอบในไฟล์ app.js ว่าเชื่อมต่อกับชื่อใด)
เเละ Collection products


2. ติดตั้งโปรเจกต์
Clone หรือ Download ไฟล์จาก GitHub นี้ลงเครื่อง

เปิด Terminal ในโฟลเดอร์โปรเจกต์แล้วรันคำสั่ง: npm install



3. เริ่มใช้งาน
รันคำสั่งเพื่อเริ่ม Server: node app.js  Or npm start
